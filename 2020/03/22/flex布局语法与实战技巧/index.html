# Flex布局语法与实战技巧

​ 布局(layout)是CSS中十分重要的一项技术，传统的布局方案，基于盒子模型，依赖display属性+position属性+float属性。在日常使用中，虽然这样的方案已经能实现大多数的需求，但是在某些特殊布局上还是有些困难（如垂直居中）。

​ Flex布局，即弹性盒子布局，是W3C提出的一种布局方案。可以简便、完整、响应式地实现各种布局。目前所有浏览器都已支持flex属性。

​ 此处对Flex布局做一个总结归纳，如不理解概念可参考阮一峰老师的《Flex 布局教程》。

# 语法

​ Flex布局包括Flex容器与Flex项目，采用Flex属性的元素成为容器，其内部所有子元素成为项目,且子元素的float、clear、vertical-align都将失效。

.box {
  display: flex;
}

​ flex布局默认存在两条轴，水平的主轴和垂直的交叉轴。

# 容器的属性

# flex-diretion

​ 决定主轴的方向（即项目的排列顺序），可选属性有row|row-reverse|column|column-reverse

# flex-wrap

​ 决定当一条轴线排不下项目时，该如何进行换行。可选属性有nowrap|wrap|wrap-reverse

​ nowrap：不换行；wrap：换行，第一行在上方；wrap-reverse：换行，第一行在下方。

# flex-flow

​ 是felx-direcion和flex-wrap的简写模式，默认值为 row wrap。

# justify-content

​ 决定在主轴上的对齐方式，可选属性有flex-start|flex-end|center|space-between|space-around，分别为左对齐、右对齐、居中、两端对齐且项目之间的距离相等、每个项目两侧的间距相等。

# align-items

​ 决定在交叉轴上的对齐方式，可选属性有flex-start|flex-end|center|stretch|baseline，分别为交叉轴的起点对齐、终点对齐、居中、如果为设置高度将沾满整个容器的高度、项目的第一行文字的基线对齐。

# align-content

​ 决定多根轴线的对齐方式，可选属性有flex-start|flex-end|center|strech|space-between|space-around

# 项目的属性

# order

​ 属性定义项目的排列顺序。数值越小，排列越靠前，默认为0。

# flex-grow

​ 属性定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大。

# flex-shrink

​ 属性定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。

# flex-basis

​ 属性定义了在分配多余空间之前，项目占据的主轴空间（main size）。浏览器根据这个属性，计算主轴是否有多余空间。它的默认值为auto，即项目的本来大小。

# flex

​ flex属性是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选。

# align-self

​ align-self属性允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。可选属性有auto | flex-start | flex-end | center | baseline | stretch。
