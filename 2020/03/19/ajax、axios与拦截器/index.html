# Ajax、Axios与拦截器

# Ajax

​ Ajax是一种用于创建动态页面的技术。全称为async javascript and XML。通俗的理解就是使用它向服务器发送请求，得到部分数据，以此更新页面。

# 基本语法

​ Ajax关键在于创建一个XMLHttpRequest对象，用于在后台与服务器交换数据。一个完整的列子如下所示：

const http = new XMLHttpRequest();
http.open(method, url, asyncFlag);//method为请求方式get或者post；url为请求路径；asyncFlag为是否为异步请求true或者false
http.send(string) //string为对象参数，仅限于Post

//向请求头添加信息 http.setRequestHeader(header,value)
//获取请求信息 http.responseText:以字符串形式返回；http.responseXML:以XML形式返回

# readyState

​ 当请求发送之后，XMLHttpRequest会有几种状态，这个值存储在readyState中。当readyState改变时，会执行onreadystatechange方法。在这个方法中 ，我们可以根据readyState的值不同做相应处理。

const http = new XMLHttpRquest();
http.onreadystatechange=function() {
  if (http.readyState==4 && http.status==200) {
    console.log('请求完毕,返回信息为' + http.responseText)；
  }
}

​ readyState的值集为0、1、2、3、4，分别代表着http请求的五种状态。

/*
0: 请求未初始化
1: 服务器连接已建立
2: 请求已接收
3: 请求处理中
4: 请求已完成，且响应已就绪
*/

# Axios

​ 如果每次发送请求都还要手动创建XMLHttpRequest对象，代码就会显得太过繁琐。Axios是一个封装的http库，用于创建XMLHttpRequest对象，且向服务器发送数据。在日常开发中，可以通过封装Axios方法，快捷方便地向服务器发送请求。

​ axios发送Get请求：

axios.get('/user?ID=12345')
  .then(function (response) {
    console.log(response);
  })
  .catch(function (error) {
    console.log(error);
  });

​ axios发送Post请求：

axios.post('/user', {
    firstName: 'Fred',
    lastName: 'Flintstone'
  })
  .then(function (response) {
    console.log(response);
  })
  .catch(function (error) {
    console.log(error);
  });

​ 更多的axios库的方法可以参考官方文档。同时，axios基于Promise,因此可以使用async/await更好的封装自己的http方法。

# 拦截器

​ 在发送请求的时候，我们可能需要统一为http请求添加请求头、鉴权等信息，也可能需要统一对http请求的响应进行处理。这时候可以用到axios的拦截器interceptors。它会在我们发送请求时或者收到响应时进行统一的处理。

import axios from 'axios';
//请求拦截器
axios.interceptors.request.use(config => {
  config.baseURL = '/api/';
  config.timeout = 6000;
  let token = storage.get('user-token');
  config.headers['Authorization'] = token;
  //统一添加了基地址、timeout时间、鉴权
})

//响应拦截器
axios.interceptors.response.use(res => {
  if (res && res.success) {
    //返回正确响应，统一进行部分处理
  };
}, err => {
  //发生错误时统一在此处进行处理
})
